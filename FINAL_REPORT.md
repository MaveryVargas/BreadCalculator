## 🚀 COMO GERAR APK

### Opção 1: EAS Cloud (Recomendado) ⭐

```powershell
# 1. Criar conta em https://expo.dev
# 2. Instalar ferramenta
npm install -g eas-cli

# 3. Login
eas login

# 4. Gerar APK
eas build --platform android --profile preview

# ⏱️  Tempo: ~10 minutos
# 📦 Resultado: APK pronto para download
```

### Opção 2: Local com Android Studio

```powershell
# Requer: Android Studio + SDK configurado
expo prebuild --clean
cd android
./gradlew assembleRelease
# 📂 Resultado: android/app/build/outputs/apk/release/
```

### Opção 3: Rápido - Expo Go

```powershell
npm install
npm start
# Escanear QR com Expo Go
# ⚠️  Não gera APK, apenas testa
```

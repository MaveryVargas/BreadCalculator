# 🍞 Calculadora de Pão Alegre

Um aplicativo Android alegre e colorido para gerenciar seu inventário de padaria!

## Recursos ✨
- ➕ Adicionar, editar e deletar itens
- 📂 Organizar por categorias (Pão, Pastel, Biscoito, Bolo, Rosquinha, Outro)
- 📊 Estatísticas em tempo real
- 🔍 Funcionalidade de pesquisa
- 😊 Interface alegre com cores quentes

## Instalação 🚀

1. Instale Node.js e npm
2. Instale Expo CLI: `npm install -g expo-cli`
3. Clone este repositório
4. Instale as dependências: `npm install`
5. Execute no Android: `npm run android`

## Cores Utilizadas 🎨

- Ouro: #FFD700

## Como Usar

1. **Adicionar Item**: Clique no botão ➕ na parte inferior direita
2. **Editar Quantidade**: Use os botões +/- em cada item
3. **Deletar Item**: Clique no botão 🗑️
4. **Pesquisar**: Use a barra de busca para filtrar itens
5. **Categorizar**: Todos os itens são organizados automaticamente

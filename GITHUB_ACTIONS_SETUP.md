# 🔧 GitHub Actions Automated Build Guide\n\n## ✅ Setup Instructions\n\n### Step 1: Create Expo Account (Optional but Recommended)\n1. Go to https://expo.dev\n2. Sign up for a free account\n3. Create an access token in your account settings\n\n### Step 2: Add EXPO_TOKEN Secret to GitHub\n1. Go to your repository settings: https://github.com/MaveryVargas/BreadCalculator/settings\n2. Click **Secrets and variables** → **Actions**\n3. Click **New repository secret**\n4. Name: `EXPO_TOKEN`\n5. Value: Paste your Expo access token\n6. Click **Add secret**\n\n### Step 3: Trigger the Build\n\n**Option A: Automatic (Recommended)**\n- Push to `main` or `develop` branch\n- Workflow runs automatically on every push\n\n**Option B: Manual Trigger**\n1. Go to: https://github.com/MaveryVargas/BreadCalculator/actions\n2. Click **🍞 Build BreadCalculator APK**\n3. Click **Run workflow**\n4. Click **Run workflow** button\n\n**Option C: Make a commit**\n```bash\ngit add .\ngit commit -m \"trigger build\"\ngit push origin main\n```\n\n---\n\n## 📊 Build Workflow Features\n\n### 🏗️ Build Job\n- Checks out your code\n- Sets up Node.js 18\n- Installs dependencies\n- Builds APK using EAS (Expo)\n- Uploads build logs as artifacts\n\n### 🧪 Test Job\n- Validates code quality\n- Runs linting checks\n- Ensures code standards\n\n### 📧 Notification Job\n- Reports build status\n- Success/Failure notifications\n\n---\n\n## 📥 Download Your APK\n\n### Via Expo Dashboard (Recommended)\n1. Go to https://expo.dev/builds\n2. Find your BreadCalculator build\n3. Click to view details\n4. Download the APK file\n\n### Via GitHub Actions\n1. Go to your repository actions: https://github.com/MaveryVargas/BreadCalculator/actions\n2. Click the latest **Build BreadCalculator APK** workflow\n3. Download artifacts (build logs)\n4. Check Expo Dashboard for APK download link\n\n---\n\n## 🚀 Build Triggers\n\nThe workflow runs automatically on:\n- ✅ Push to `main` branch\n- ✅ Push to `develop` branch\n- ✅ Pull requests to `main` or `develop`\n- ✅ Manual trigger via Actions tab\n\n---\n\n## ⚙️ Customization\n\nEdit `.github/workflows/build-apk.yml` to:\n- Change Node.js version\n- Add additional build steps\n- Configure different build types\n- Add environment variables\n\n---\n\n## 🔑 Without EXPO_TOKEN\n\nIf you don't set up an Expo account:\n- Workflow will still run\n- Local build may require Android SDK setup\n- Refer to Method 2 in EAS_BUILD.md\n\n---\n\n## 📱 Install APK on Android\n\n```bash\n# Transfer APK to your device\nadb install BreadCalculator.apk\n\n# Or enable installation from unknown sources and open APK on device\n```\n\n---\n\n## ❓ Troubleshooting\n\n### Build fails with \"EXPO_TOKEN not set\"\n- Add EXPO_TOKEN secret (see Step 2 above)\n\n### Build takes too long\n- Normal! EAS builds can take 10-30 minutes\n- Check Expo Dashboard for real-time status\n\n### APK won't install\n- Ensure minimum Android version requirements are met\n- Check device storage space\n- Enable installation from unknown sources\n\n---\n\n## 📚 Learn More\n\n- [Expo Build Documentation](https://docs.expo.dev/build/setup/)\n- [EAS Build Guide](https://docs.expo.dev/build/introduction/)\n- [GitHub Actions Documentation](https://docs.github.com/en/actions)\n\n---\n\n## 🎉 Success!\n\nYour BreadCalculator app is now set up for automated builds! 🍞✨\n"
